# 25/75押し目専用 Pro v2 (Winning Edition)

25日EMA・75日EMAを軸にした押し目買いインジケーターの改良版です。
旧版 (Pro v1) の課題であった **勝率・リスクリワード比** を大幅に改善しました。

---

## コンセプト

> **上昇トレンド中に25日EMAまで押した銘柄を、反発の初動で拾い、トレーリングストップで利益を守る**

---

## v1 → v2 の主な変更点

| カテゴリ | v1 | v2 (Winning Edition) |
|---------|-----|-----|
| トレンド強度 | EMAの並び順のみ | **ADX (≥20)** を追加し弱トレンドを排除 |
| 利確 | 25EMA割れまで保有 | **ATRベース TP1/TP2** + トレーリングストップ |
| 損切 | なし（25EMA割れがEXIT） | **ATRベース損切ライン**を自動設定 |
| 利益保護 | なし | **ATRトレーリングストップ** (TP1後に引き締め) |
| エントリーパターン | 5EMAリクレーム+前日高値 | ハンマー足・包み足・**BBスクイーズブレイク**を追加 |
| RSI | 単純反転 | **隠れ強気ダイバージェンス**を検出 |
| 出来高 | 出来高倍率のみ | **出来高枯れ→急増パターン**を高品質シグナルとして加点 |
| EMA | 5/25/75 | 5/25/75 + **200日EMA** (上位トレンドフィルター) |
| EXIT | 25EMA割れ・MACD悪化 | 4段階: WARN → TRAIL EXIT → SL → モメンタム崩壊 |
| パネル | 基本情報のみ | 損益%・ADX・BBスクイーズ・トレーリング値を追加 |

## 使用する指標

| 指標 | 用途 |
|------|------|
| EMA 5 / 25 / 75 / 200 | トレンド方向・押し目ゾーン・上位トレンド |
| RSI (14) | 反転検出・ダイバージェンス・過熱判定 |
| MACD (12, 26, 9) | モメンタム転換の確認 |
| ADX (14) | トレンドの「強さ」を数値で判定 |
| ATR (14) | ボラティリティ計測・TP/SL/トレーリングの算出 |
| ボリンジャーバンド (20, 2σ) | スクイーズ（収縮）→ ブレイクアウトの検出 |
| 出来高 20日平均比 | 出来高の質（枯れ→急増パターン） |
| 週足 EMA10 | 週足レベルのトレンド方向フィルター |
| VWAP | オプションの出来高加重価格フィルター |

## シグナル一覧

### エントリー

| シグナル | 条件 | 表示 |
|---------|------|------|
| **BUY** | 押し目条件成立 + スコア ≥ 基準 | 緑ラベル (足の下) |
| **A+ BUY** | BUY + スコア ≥ 強シグナル基準 + 低ボラ | 大きい緑ラベル (足の下) |
| **BUY HALF** | BUY + 高ボラティリティ環境 | オレンジラベル (足の下) |

### 3つのエントリーパターン

1. **押し目IN** — 25EMAタッチ → 5EMAリクレーム / ハンマー / 包み足
2. **再IN** — 上昇継続中の25EMA付近からの再加速
3. **スクイーズブレイク** — BBスクイーズ収縮 → 拡大の初動

### EXIT系

| シグナル | 条件 | 表示 |
|---------|------|------|
| **TP1** | ATR × 2.0倍到達 | 緑フラグ |
| **WARN** | 5EMA割れ or RSIモメンタム低下 | 黄色ラベル |
| **EXIT** | 25EMA割れ / MACD悪化 / 過熱利確 | 赤ラベル |
| **TRAIL EXIT** | トレーリングストップヒット | オレンジラベル |
| **SL** | 損切ライン到達 | 赤×マーク |

## スコアリングシステム (最大約30点)

| 項目 | 最大点 | 説明 |
|------|--------|------|
| トレンド (EMA並び順) | 3 | 5>25>75 の並び順 |
| 強トレンドボーナス | 2 | パーフェクトオーダー + ADX≥25 |
| 週足方向 | 2 | 週足EMA10の上 |
| EMA200上 | 1 | 200日線の上にいるか |
| ADX | 2 | ≥25: 2点, ≥20: 1点 |
| 乖離ゾーン | 3 | 25日乖離が押し目レンジ内 |
| 25EMAタッチ | 2 | 直近5本で25EMAにタッチ |
| リクレーム | 2 | 5EMA回復 / 前日高値ブレイク / ハンマー |
| RSI反転 | 2 | RSIが下から反転 |
| RSIダイバージェンス | 2 | 隠れ強気ダイバージェンス |
| MACD | 3 | ゴールデンクロス: 3, ヒストグラム改善: 1-2 |
| ローソク足 | 2 | 包み足: 2, 陽線: 1 |
| 出来高 | 3 | 枯れ→急増: 3, 1.5倍: 2, 1.2倍: 1 |
| スクイーズブレイク | 2 | BBスクイーズからの脱出 |
| リスク減点 | -5 | 高ボラ / 過大乖離 / 急落 / EMA200下 |

**デフォルト基準: 通常 8点以上 / 強シグナル 12点以上 / 高ボラ時 10点以上**

## レジーム判定

| レジーム | 条件 | 背景色 | 動作 |
|---------|------|--------|------|
| 急落 | 日次変動≤-4.5% & 25EMA下 | 赤 | シグナル停止 |
| 下降 | 75EMA下 or 75EMA傾き<0 | 暗赤 | シグナル停止 |
| 強上昇 | パーフェクトオーダー & ADX≥25 | 緑 | 通常シグナル |
| 高ボラ | ATR%≥2.8 or 平常比≥1.35倍 | オレンジ | HALF BUY + スコア+2 |
| 通常 | 上記以外 | 薄緑 | 通常シグナル |

## 利確・損切の仕組み

```
エントリー価格
  ├─ TP1 = エントリー + ATR × 2.0  → 到達でトレーリング引き締め
  ├─ TP2 = エントリー + ATR × 4.0  → 到達でEXIT
  ├─ SL  = エントリー - ATR × 1.8  → 到達で損切
  └─ Trail = 現在値 - ATR × 2.5    → 上方向のみ更新（TP1後は×0.6に引き締め）
```

## 推奨設定

- **日足チャート**での使用を想定
- デフォルト設定のまま使用可能
- VWAPフィルターは日中足向け（日足ではOFF推奨）
- 高ボラ銘柄では自動的にHALF BUYとスコア基準引き上げが適用される
